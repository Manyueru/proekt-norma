# Проект «Норма»

Личная образовательная платформа по развитию ребёнка, логопедии, нейропсихологии, специальной педагогике и клинической насмотренности.

## Важно

Сайт предназначен для обучения и не ставит медицинские диагнозы. Часть содержимого первой версии является демонстрационной и помечена как требующая проверки.

## Как запустить на компьютере

1. Установить актуальную LTS-версию Node.js.
2. Открыть папку проекта в терминале.
3. Выполнить `npm install`.
4. Выполнить `npm run dev`.
5. Открыть адрес, который появится в терминале, обычно `http://localhost:3000`.

## Проверка перед публикацией

- `npm run lint` — проверка TypeScript.
- `npm run build` — производственная сборка.
- `npm audit` — проверка известных уязвимостей зависимостей.

## Как добавить учебную тему

Учебные темы находятся в `data/topics.json`. Новая запись должна соответствовать типу `Topic` из `lib/types.ts`. Для страницы темы используется поле `slug`; оно должно быть уникальным и написано латиницей через дефисы.

## Как добавить источник

Источники находятся в `data/sources.json`. Не добавляйте вымышленные ссылки. Если ссылка ещё не проверена, оставьте `url` пустым и явно пометьте описание как демонстрационное.

## Как добавить видео

Видео находятся в `data/videos.json`. Пустая ссылка не отображается как рабочая. До проверки видео не указывайте вымышленные тайм-коды.

## Конспекты и прогресс

Конспекты и прогресс сохраняются только в `localStorage` текущего браузера. Они не синхронизируются между устройствами и могут исчезнуть после очистки данных браузера.

На странице «Конспекты» доступны:

- экспорт резервной копии в JSON;
- импорт резервной копии;
- полный сброс локальных данных с подтверждением.

Перед очисткой браузера или переходом на другое устройство экспортируйте резервную копию.

## Публикация

Проект можно разместить на Vercel. Проще всего загрузить файлы в GitHub-репозиторий, импортировать репозиторий в Vercel и нажать Deploy.

## Текущие ограничения

- нет регистрации и серверной базы данных;
- данные не синхронизируются между устройствами;
- новые учебные материалы добавляются через JSON-файлы;
- часть демонстрационных источников требует библиографической проверки;
- сайт не заменяет очную диагностику и консультации специалистов.
