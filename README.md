# Проект «Норма»

Персональная обучающая платформа для будущего логопеда. Логопедия является основной профессиональной траекторией; онтогенез, дизонтогенез, ОВЗ, нейропсихология, клинические основы и диагностика образуют обязательный расширяющий фундамент.

## Что работает

- адаптивный интерфейс для компьютера и телефона;
- 22 учебных модуля и 251 тема;
- безопасное разделение проверенного содержания, черновиков и тем-каркасов;
- карта обязательных компетенций и контроль пробелов;
- тренажёр клинической и педагогической насмотренности;
- клинические задачи;
- режим изучения книг по блокам с заметками и тестами;
- атлас российских и зарубежных методик с оценкой доказательной базы и ограничений;
- автоматическая лента новых публикаций PubMed;
- растущий словарь, источники и видеотека;
- локальный кабинет, прогресс, конспекты, экзамены и дедлайны;
- выбор тем через поиск и фильтр по модулю вместо длинного списка;
- общий поиск по темам, источникам, кейсам, словарю, книгам и методикам;
- экспорт и импорт локальной резервной копии версии 4.

## Важное ограничение

Персональные данные пока сохраняются в `localStorage` конкретного браузера. Между устройствами они не синхронизируются. Резервная копия версии 4 включает:

- прогресс учебных тем;
- конспекты;
- ответы на клинические задачи;
- результаты тестов;
- экзамены и дедлайны;
- разборы насмотренности;
- прогресс и заметки по книгам.

## Научная лента

Страница `/research` получает метаданные новых публикаций из PubMed через официальный API NCBI. Записи автоматически имеют статус «новая, не проверена». Они не изменяют учебные темы, пока публикация не будет отдельно проанализирована.

Запрос кешируется на 6 часов, чтобы не обращаться к внешнему сервису при каждом открытии страницы.

## Локальный запуск

Требуется Node.js 22.

```bash
npm ci
npm run dev
```

Проверка TypeScript:

```bash
npm run lint
```

Производственная сборка:

```bash
npm run build -- --webpack
```

## Основные данные

- `data/modules.json` — 22 модуля;
- `data/topics.json` — 251 тема;
- `data/observations.json` — случаи для насмотренности;
- `data/books.json` — маршруты изучения книг;
- `data/methods.json` — атлас методик;
- `data/sources.json` — каталог источников и их статусы;
- `data/cases.json` — клинические задачи;
- `data/glossary.json` — словарь;
- `data/videos.json` — видеотека.

## Публикация

Пакет предназначен для одной объединённой загрузки после просмотра локально. Netlify не использовался при разработке этой версии, поэтому кредиты на промежуточные сборки не расходовались.
